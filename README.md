# Heart Disease Prediction using Machine Learning

## Overview

This project builds a machine learning model to predict the presence of heart disease based on patient health parameters. It is a binary classification problem aimed at assisting in early diagnosis and risk assessment.

---

## Dataset Information

* Dataset: Heart Disease Dataset
* Features: Medical and physiological attributes
* Target:

  * 0 → No Heart Disease
  * 1 → Heart Disease

### Feature Description

* Age
* Sex
* Chest Pain Type
* Resting Blood Pressure
* Cholesterol
* Fasting Blood Sugar
* Resting ECG Results
* Maximum Heart Rate Achieved
* Exercise Induced Angina
* ST Depression
